## 🛡️ Troubleshooting

If the software fails to draw the overlay, check these common items:

* Administrator Privileges: The program must run with administrative rights to read game data. If you skip this, the data fields will remain blank.
* Antivirus Software: Sometimes security programs flag external overlays. If the program fails to start, check your antivirus quarantine folder and add an exclusion for the Meccha Chameleon installation path.
* Overlay Conflicts: Close other programs that draw on your screen, such as Discord overlays, game bars, or screen recording software, to prevent technical conflicts.
* DirectX Version: Ensure the game uses DirectX 11. If you run the game in Vulcan or OpenGL mode, the overlay will not work.
